# Wedding Scanner

A web app that lets users enter wedding-related categories and searches Alibaba/AliExpress for matching products.

## Structure

```
wedding-scanner/
├── client/   # React + TypeScript (Vite) frontend
└── server/   # Node.js + Express + TypeScript backend
```

## Getting Started

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd server
npm run dev
```
